🌐 Network Topology Project – Jakarta, Bandung, Yogyakarta
📌 Deskripsi

Project ini merupakan simulasi topologi jaringan yang menghubungkan tiga wilayah, yaitu Jakarta, Bandung, dan Yogyakarta menggunakan Cisco Packet Tracer. Setiap wilayah memiliki segmentasi jaringan tersendiri dengan implementasi berbagai teknologi jaringan untuk mensimulasikan lingkungan enterprise.

🧩 Topologi Jaringan

Topologi terdiri dari:

3 Router (masing-masing mewakili Jakarta, Bandung, Yogyakarta)
Beberapa Switch di setiap wilayah
End devices (PC/Laptop)
Wireless Access Point untuk koneksi WLAN
⚙️ Konfigurasi yang Digunakan
1. VLAN (Virtual Local Area Network)
Digunakan untuk segmentasi jaringan di setiap wilayah
Memisahkan traffic berdasarkan departemen atau fungsi
Meningkatkan keamanan dan efisiensi jaringan
2. WLAN (Wireless Local Area Network)
Implementasi Access Point untuk koneksi wireless
Client dapat terhubung tanpa kabel dalam satu jaringan
3. DHCP (Dynamic Host Configuration Protocol)
IP address diberikan secara otomatis ke client
Mengurangi konfigurasi manual dan meminimalisir kesalahan
4. OSPF (Open Shortest Path First)
Digunakan sebagai routing protocol antar wilayah
Memungkinkan komunikasi antar jaringan Jakarta, Bandung, dan Yogyakarta
Menggunakan konsep dynamic routing
5. Port Security
Diterapkan pada switch untuk membatasi akses perangkat
Mencegah perangkat tidak dikenal terhubung ke jaringan
Meningkatkan keamanan layer 2
🎯 Tujuan Project
Memahami implementasi jaringan skala multi-site
Mengaplikasikan VLAN untuk segmentasi jaringan
Menggunakan OSPF untuk routing dinamis antar wilayah
Menerapkan DHCP untuk efisiensi IP management
Meningkatkan keamanan jaringan dengan Port Security
