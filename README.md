# 🌐 Network Topology Project – Jakarta, Bandung, Yogyakarta

## 📌 Deskripsi
Project ini merupakan simulasi topologi jaringan yang menghubungkan tiga wilayah, yaitu Jakarta, Bandung, dan Yogyakarta menggunakan Cisco Packet Tracer. Setiap wilayah memiliki segmentasi jaringan tersendiri dengan implementasi berbagai teknologi jaringan untuk mensimulasikan lingkungan enterprise.

## 🧩 Topologi Jaringan
Topologi terdiri dari:
- 3 Router (Jakarta, Bandung, Yogyakarta)
- Beberapa Switch di setiap wilayah
- End devices (PC/Laptop)
- Wireless Access Point (WLAN)

## ⚙️ Konfigurasi yang Digunakan

### VLAN (Virtual Local Area Network)
- Segmentasi jaringan berdasarkan kebutuhan
- Memisahkan traffic antar departemen
- Meningkatkan efisiensi dan keamanan

### WLAN (Wireless Local Area Network)
- Implementasi access point untuk koneksi wireless
- Client dapat terhubung tanpa kabel

### DHCP (Dynamic Host Configuration Protocol)
- Pemberian IP address secara otomatis
- Mengurangi konfigurasi manual

### OSPF (Open Shortest Path First)
- Routing dinamis antar wilayah
- Menghubungkan jaringan Jakarta, Bandung, dan Yogyakarta

### Port Security
- Membatasi perangkat yang dapat terhubung ke switch
- Meningkatkan keamanan jaringan layer 2

## 🎯 Tujuan Project
- Memahami jaringan multi-site
- Mengimplementasikan VLAN dan routing dinamis
- Menggunakan DHCP untuk efisiensi IP
- Meningkatkan keamanan dengan port security
